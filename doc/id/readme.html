Video to MP3 Converter

Selamat datang di add-on Video to MP3 Converter. Alat ini dirancang untuk memudahkan Anda mengekstrak audio dari berbagai format video secara cepat dan mudah langsung dari File Explorer.

Cara Penggunaan

  1. Buka File Explorer dan pilih file video yang ingin dikonversi.
  2. Tekan Ctrl + C untuk menyalin file tersebut ke clipboard.
  3. Tekan perintah pintas add-on (lihat tabel di bawah).
  4. Tunggu suara Beep pertama dan pesan "Memproses file".
  5. Setelah selesai, Anda akan mendengar Beep nada tinggi dan pesan "Konversi selesai".

Daftar Perintah Pintas

Fungsi Perintah Pintas
Konversi Video ke MP3 NVDA + Alt + C

Fitur Unggulan

  • Tanpa Jendela CMD: Proses berjalan di latar belakang tanpa mengganggu layar.
  • Notifikasi Audio: Menggunakan suara beep yang intuitif untuk menandakan status proses.
  • Kualitas Tinggi: Audio dikonversi dengan bitrate 192kbps yang jernih.
  • Nama File Bersih: File hasil konversi menggunakan nama yang sama dengan video aslinya.
