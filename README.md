# Audio Extractor for NVDA

Audio Extractor is a simple and efficient add-on for the NVDA screen reader that allows users to extract audio from video files using a clipboard-based workflow.

## Key Features
- **Fast & Lightweight:** Extract audio without opening heavy video editing software.
- **Clipboard Integration:** Simply copy a video file and run the command.
- **Output Formats:** Supports conversion to **MP3** and **WAV**.
- **Fully Accessible:** Specifically designed for NVDA users.

## How to Use
1. Select a video file in File Explorer.
2. Press **Ctrl + C** to copy the file.
3. Press the shortcut: **NVDA + Alt + V**.
4. Choose the desired output format (**MP3** or **WAV**).
5. The extracted audio will appear in the same folder as the original video.

# Audio Extractor untuk NVDA (Bahasa Indonesia)

**Audio Extractor** adalah add-on NVDA yang memudahkan pengguna mengambil suara dari file video dengan sangat praktis.

## Fitur Utama
- **Cepat:** Tidak perlu aplikasi berat.
- **Sistem Salin-Tempel:** Cukup tekan **Ctrl + C** pada video, lalu jalankan shortcut.
- **Format:** Mendukung **MP3** dan **WAV**.

## Cara Pakai
1. Pilih file video di File Explorer.
2. Tekan **Ctrl + C**.
3. Tekan shortcut: **NVDA + Alt + V**.
4. Pilih format **MP3** atau **WAV**.
5. Hasil audio akan muncul di folder yang sama.
