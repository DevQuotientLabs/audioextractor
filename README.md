# Audio Extractor for NVDA

[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)
[![NVDA Version](https://img.shields.io/badge/NVDA-2021.1+-green.svg)](https://www.nvaccess.org/)

**Audio Extractor** is a professional NVDA add-on developed by **DevQuotiontLab** that allows users to quickly extract audio tracks from video files directly from the file explorer.

## 🚀 Key Features
- **Fast Extraction:** High-performance audio extraction powered by FFmpeg.
- **Multiple Formats:** Supports various output audio formats.
- **Accessible UI:** Fully compatible with screen readers with clear audio feedback.
- **Background Processing:** Runs in a separate thread so it won't freeze your NVDA.

## 🛠️ How to Use
1. Select a video file in your file explorer.
2. Press `Ctrl + C` to copy the file.
3. Press the shortcut `NVDA + Alt + V`.
4. Choose your desired format in the dialog and press **Extract**.

## 📦 Requirements
- NVDA 2021.1 or later.
- `ffmpeg.exe` (Bundled with the add-on package in the `globalPlugins/video_converter/` directory).

## 📥 Installation
1. Download the latest `.nvda-addon` file from the [Releases](https://github.com/DevQuotiontLab/audioExtractor/releases) page.
2. Open the file to install it into NVDA.
3. Restart NVDA if prompted.
